## 概念
“占卜大战”是一个基于SCP作品的2D多人卡牌游戏。游戏中，玩家们通过不同的占卜方式和功能卡牌进行对战，最终以城市中最多NPC数量为胜利。

目前计划如下：
* 版本为Python3
* 基于Pygame进行编写
* 2D多人联机游戏

## 框架
* [Pygame](https://www.pygame.org/news) - 游戏主体，多媒体
* Socket - 联网模块
